How to run:

Needs Remix IDE to compile and deploy.

Import your .sol file to Remix

Compile, in the left bar, the .sol file

Deploy, in the left bar, the .sol file with the parameters _dexSwapRate, _paymentCycle, _interestRate, _terminationFee and _maxLoanDuration. Click then on the button transact

Go to Deployed contracts and open you deployed contract and start making the operations.

Hint: Add 10000 Wei(or another amount, your choice) at the beginning and then go to the bottom of deployed contracts and click transact to have sufficient amount to make simple operations.
